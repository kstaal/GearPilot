# Gear Pilot Changelog

## [1.0.0] - 2026-03-10

### Added
- Initial release
- Custom tooltip values display for items
- Hover over any item to see custom calculated values
- Slash commands: `/gp` or `/gearpilot`
- Toggle addon on/off with `/gp toggle`
- Toggle proof string display with `/gp proof`
- Support for bag items, bank items, and hyperlinked items

### Features
- Displays custom "GP Proof" value in item tooltips
- Green-colored text for easy visibility
- Configurable through saved variables
- Chat command interface for easy control
